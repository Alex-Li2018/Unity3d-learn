# 循环语句

## while语句

```csharp
int a = 10;

while(a < 20) {
    a++;
    // 从11开始输出
    Console.writeLine(a);
}
```

## do...while语句
```csharp
int a = 10;

do {
    // 从10开始输出
    Console.writeLine(a); 
    a++;
} while(a < 20)
```

## break/continue语句

- break 终止loop与switch语句，程序流将继续执行紧接着的loop或swutch的下一条语句
- continue 跳过本轮循环，开始下一轮循环
